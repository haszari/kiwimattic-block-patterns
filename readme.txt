== Description ==

This is a demo repo to scaffold setup for creating a new block patterns plugin

== Installation ==

- Check out the repo
- Run `npm install`
- Run `npm run start-wp` and you should then have a WP instance running at http://localhost:8888/wp-admin/, username `admin` password `password`, and with the kiwimattic-block-patterns plugin installed and activated.
- To stop WP run `npm run stop-wp`
- To build block patterns and translations run `npm run start`
